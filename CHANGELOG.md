# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.0.6] — 2026-07-22

- импорт библиотеки из ВКонтакте: вход через OAuth, треки тянутся из `audio.get`
- поиск переписан по логике ВК
- liquid glass в оформлении, новый таббар
- обновлённые скриншоты в `docs/screens`
- релиз собирается в GitHub Actions по тегу `v*`

APK подписан новым ключом, поверх 1.0.5 он не установится — старую версию нужно удалить.

## [1.0.5] — 2026-07-19

- `release.js` копирует `www` в android перед сборкой APK
- убрал `Kvinta.bat`, README подтянут под релизы

## [1.0.4] — 2026-07-18

- артисты трека по отдельности: кликабельные ссылки в списках и плеере, попап выбора, когда их несколько

## [1.0.3] — 2026-07-18

- страница альбома
- кастомный тайтлбар и свои тултипы
- живая перемешка при повторе
- переход к артисту прямо из плеера, обложки в `covers`

## [1.0.2] — 2026-07-18

- меню трека с лайком и скачиванием
- сброс настроек
- переход к артисту из плеера

## [1.0.1] — 2026-07-18

- поиск по артистам и альбомам, карточки артистов
- починил создание плейлистов
- обновления работают без токена, кастомный установщик
- чистка кода

## [1.0.0] — 2026-07-17

Первая версия: автообновление, сборка установщика и APK.
