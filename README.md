# Kvinta

Локальный музыкальный сервис в красном стиле. Избранное, плейлисты, загрузки и настройки хранятся только на этом компьютере.

© MortisClub 2026

## Запуск

Дважды кликни `Kvinta.bat` — при первом запуске сам поставит зависимости.

## Возможности

- Чарт, новые релизы, поиск, персональные подборки «Для тебя», история «Ты недавно слушал»
- Избранное и свои плейлисты (хранятся только локально)
- Загрузка треков для офлайна (папка `Музыка\Kvinta`)
- 10-полосный эквалайзер с пресетами, скорость, баланс, моно, нормализация, плавные переходы
- Таймер сна (15/30/60/90 мин или до конца трека)
- Медиаклавиши клавиатуры, управление с экрана блокировки (Media Session)

## Мобильная версия (`mobile/`)

Capacitor-обёртка над тем же кодом: `node mobile/sync.js` → открыть `mobile/android` в Android Studio и собрать APK.

Иконки приложения (Android mipmap/adaptive, `assets/icon.png`, `.ico` для Electron) генерируются командой `node tools/make-icons.js`.

- Нижняя таб-навигация, компактный мини-плеер, полноэкранный «Сейчас играет» (тап по мини-плееру, свайп вниз — закрыть)
- Полные настройки звука: эквалайзер, баланс, моно, нормализация, preamp, фейды — трек с обработкой загружается целиком (нативно, мимо CORS) и играет из blob
- Громкость и таймер сна в настройках, управление из шторки/с экрана блокировки
